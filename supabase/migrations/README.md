# Database migrations

These files are the source of truth for the Supabase schema (project
`uxqtbbfubvlkkynnlyfa`). They were recovered from the live database's applied
migration history so the schema is reproducible from version control.

Apply with the Supabase CLI:

```bash
supabase link --project-ref uxqtbbfubvlkkynnlyfa
supabase db push
```

When making schema changes, add a new timestamped file here (and apply it) rather
than editing an existing migration.
